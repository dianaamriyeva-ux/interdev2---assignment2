.・゜゜・Design Rationale・゜゜・．

I wanted to create something with the use of p5.js because it is a very interesting creative JavaScript libruary. I decided to create a web page where you can draw pixel art and easily save it. Luckily, p5.js already had templates and tutorials on creating pixels and canvases. I successfully united these two things into one and added some tools for customization!
